# /aria:pathway - Regulatory Pathway Analysis

## Purpose

Compare regulatory submission pathways across target markets (FDA, EU MDR, MFDS).

## Status

⚠️ **Under Development**

This command is planned for future release.

## Planned Features

- FDA pathway analysis (510(k), PMA, De Novo, Exempt)
- EU MDR CE marking routes
- MFDS certification/approval types
- Cross-market pathway comparison
- Recommended pathway with rationale

---

For current functionality, please use `/aria:determine` or `/aria:classify`.
