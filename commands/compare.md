# /aria:compare - Regulatory Comparison

## Purpose

Compare regulatory requirements across multiple countries (FDA, EU MDR, MFDS).

## Status

⚠️ **Under Development**

This command is planned for future release.

## Planned Features

- Cross-country comparison matrix
- Similarities and differences
- Harmonized standards identification
- Strategic recommendations

---

For current functionality, please use `/aria:determine` or `/aria:classify`.
