# /aria:chat - Regulatory Q&A

## Purpose

Free-form conversation for regulatory questions with automatic skill routing.

## Status

⚠️ **Under Development**

This command is planned for future release. Current capabilities:
- Use `/aria:determine` for device determination
- Use `/aria:classify` for classification

## Planned Features

- Natural language regulatory questions
- Automatic skill detection and routing
- Conversational clarification
- Multi-turn dialogue support
