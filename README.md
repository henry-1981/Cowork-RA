# Cowork-RA

Regulatory Affairs 도메인을 위한 Anthropic Cowork 플러그인 마켓플레이스입니다.

## 플러그인

| 플러그인 | 설명 | 주요 기능 |
|----------|------|----------|
| [aria](./aria/) | AI Regulatory Intelligence Assistant - 의료기기 규제 준수 (FDA, EU MDR, MFDS) | 의료기기 판정, 등급 분류, 규제 경로 선택, 비용/일정 추정, 규제 계획 수립, 다국가 비교, 규제 브리핑 |

## 설치

Cowork에서 이 마켓플레이스를 등록하면 포함된 플러그인을 사용할 수 있습니다.

## 버전 정책

- ARIA 버전 정책 문서: `aria/VERSIONING.md`
- 자동 검증 워크플로: `.github/workflows/version-policy.yml`
